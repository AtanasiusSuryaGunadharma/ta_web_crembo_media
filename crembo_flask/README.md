# Crembo Flask App

Project baru untuk menggabungkan Flask dengan frontend mockup di folder `Mockup_hifi`.

## Run

Dari folder ini, jalankan:

```powershell
& "c:\xampp\htdocs\ta_crembo_media\.venv\Scripts\python.exe" app.py
```

Lalu buka:

`http://127.0.0.1:5000/`
