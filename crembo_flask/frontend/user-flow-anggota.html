User Flow Black Box & SUS - Anggota

Halaman ini menggabungkan black box testing role Anggota dengan SUS post-redesign. Penguji menjalankan fitur publik/no-login terlebih dahulu, lalu login sebagai anggota untuk menguji dashboard, tugas, monitoring, evaluasi, peminjaman, form pendaftaran, sertifikat, dan log pribadi.

Super Admin Admin Anggota
1. Identitas 2. Black Box Role 3. SUS 4. Pertanyaan Tambahan 5. Rekap

Bagian 1 - Identitas Penguji

Diisi sebelum menjalankan task scenario
Petunjuk revisi: Jalankan black box testing per role terlebih dahulu. Setiap butir uji diberi status Valid atau Tidak Valid. Setelah seluruh skenario role selesai, responden mengisi SUS post-redesign dan pertanyaan tambahan.

Bagian 2 - Black Box Testing Berdasarkan Role

24 card scenario • frontend-only
Skenario dibagi per lingkup fitur. Satu section dapat berisi satu card atau beberapa card jika lingkup fiturnya terdiri dari beberapa halaman, misalnya alur lupa password: forgot password → OTP verification → reset password.
