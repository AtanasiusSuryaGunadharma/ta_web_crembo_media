Bagian 3 - System Usability Scale (SUS) Akhir

Diisi setelah menyelesaikan task scenario
Berikan penilaian berdasarkan pengalaman menggunakan website Crembo Media setelah redesign. Nilai 1 berarti Sangat Tidak Setuju dan nilai 5 berarti Sangat Setuju.
1 = Sangat Tidak Setuju5 = Sangat Setuju
Pernyataan12345
1. Saya rasa saya akan sering menggunakan website Crembo Media ini.
2. Saya merasa website Crembo Media ini terlalu rumit untuk digunakan.
3. Saya merasa website Crembo Media ini mudah digunakan.
4. Saya membutuhkan bantuan teknis dari orang lain untuk dapat menggunakan website Crembo Media ini.
5. Saya merasa berbagai fitur pada website Crembo Media ini berjalan dengan baik dan saling terintegrasi.
6. Saya merasa ada terlalu banyak hal yang tidak konsisten pada website Crembo Media ini.
7. Saya membayangkan sebagian besar orang akan cepat belajar menggunakan website Crembo Media ini.
8. Saya merasa website Crembo Media ini sangat rumit untuk digunakan.
9. Saya merasa percaya diri saat menggunakan website Crembo Media ini.
10. Saya perlu mempelajari banyak hal terlebih dahulu sebelum dapat menggunakan website Crembo Media ini dengan baik.
