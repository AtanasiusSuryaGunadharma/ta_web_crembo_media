Struktur Proyek Flask Crembo Media

Dokumentasi lampiran laporan - versi ringkas dan siap cetak

Struktur proyek memisahkan file aplikasi utama, file SQL basis data, folder frontend, folder upload, aset gambar, dan file halaman HTML. Pemisahan ini membantu proses pemeliharaan karena setiap modul memiliki file antarmuka yang mudah ditemukan.

Keterangan: Fokus utama struktur di atas adalah pemisahan backend Flask, SQL database, frontend, dan file upload agar proses pengembangan, pemeliharaan, serta pelacakan modul lebih terstruktur.

Catatan: Untuk PDF, gunakan tombol "Cetak / Simpan PDF" lalu pilih printer "Save as PDF".
